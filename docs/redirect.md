# redirect

Set Headers location and exit the script.

Version : 1.0.0  
Since : 1.0.0  

---

```php
redirect(string $destination[, bool $exit = true])
```

## Parameters

- **destination**  
redirect url

- **exit**  
Default True, Exit or not after redirect


## Return Values

-


## Examples

```php
redirect("http://osw3.net"); // open osw3.net
```
